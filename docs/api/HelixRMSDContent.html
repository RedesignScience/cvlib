HelixRMSDContent¶

class cvpack.HelixRMSDContent(residues, numAtoms, thresholdRMSD=Quantity(value=0.08, unit=nanometer), halfExponent=3, normalize=False)[source]¶

The alpha-helix RMSD content of a sequence of n residues [3]:

\[\alpha_{\rm rmsd}({\bf r}) = \sum_{i=1}^{n-5} B_m\left( \sqrt{\frac{1}{30} \sum_{j=1}^{30} \left\| \hat{\bf r}_j({\bf g}_i) - {\bf A}({\bf g}_i)\hat{\bf r}_j({\bf g}_{\rm ref}) \right\|^2} \right)\]

where \({\bf g}_i\) represents a group of 30 atoms selected from residues \(i\) to \(i+5\) of the sequence, \({\bf g}_{\rm ref}\) represents the same 30 atoms in an ideal alpha-helix configuration, \(\hat{\bf r}_j({\bf g})\) is the position of the \(j\)-th atom in a group \({\bf g}\) relative to the group’s center of geometry (centroid), \({\bf A}({\bf g})\) is the rotation matrix that minimizes the RMSD between \({\bf g}\) and \({\bf g}_{\rm ref}\), and \(B_m(x)\) is a smooth step function given by

\[B_m(x) = \frac{1}{1 + x^{2m}}\]

where \(m\) is an integer parameter that controls its steepness.

Each group \({\bf g}_i\) is formed by the N, \({\rm C}_\alpha\), C, and O atoms of the backbone, as well as the \({\rm C}_\beta\) atoms of the six consecutive residues starting from residue \(i\). In the case of glycine, the missing \({\rm C}_\beta\) is replaced by the corresponding H atom.

Note

The residues must be a contiguous sequence from a single chain, ordered from the N- to the C-terminus. The maximum supported number of residues is 1024.

This collective variable was introduced in Ref. [3] with a slightly different step function. The ideal alpha-helix configuration is the same used in PLUMED v2.8.1 for its collective variable ALPHARMSD. By setting \({\rm NN}=2m\) and \({\rm MM}=4m\), PLUMED’s ALPHARMSD will match the collective variable implemented here.

Optionally, this collective variable can be normalized to the range \([0, 1]\).

Warning

Periodic boundary conditions are not supported.

Parameters:

  • residues (Sequence[Residue]) – The residues to be used in the calculation.

  • numAtoms (int) – The total number of atoms in the system (required by OpenMM).

  • thresholdRMSD (Quantity | Real) – The threshold RMSD value for the step function.

  • halfExponent (int) – The parameter \(m\) of the step function.

  • normalize (bool) – Whether to normalize the collective variable to the range \([0, 1]\).

Example

>>> import cvpack
>>> import openmm
>>> from openmm import app, unit
>>> from openmmtools import testsystems
>>> model = testsystems.LysozymeImplicit()
>>> residues = [
...     r
...     for r in model.topology.residues()
...     if 59 <= r.index <= 79
... ]
>>> print(*[r.name for r in residues])
LYS ASP GLU ... ILE LEU ARG
>>> helix_content = cvpack.HelixRMSDContent(
...     residues, model.system.getNumParticles()
... )
>>> model.system.addForce(helix_content)
6
>>> platform = openmm.Platform.getPlatformByName('Reference')
>>> integrator = openmm.VerletIntegrator(0)
>>> context = openmm.Context(model.system, integrator, platform)
>>> context.setPositions(model.positions)
>>> print(helix_content.getValue(context, digits=7))
15.915198 dimensionless

getEffectiveMass(context, digits=None)¶

Compute the effective mass of this collective variable at a given openmm.Context.

The effective mass of a collective variable \(q({\bf r})\) is defined as [1]:

\[m_\mathrm{eff}({\bf r}) = \left( \sum_{i=1}^N \frac{1}{m_i} \left\| \frac{dq}{d{\bf r}_i} \right\|^2 \right)^{-1}\]

Optionally, effective mass of this collective variable can be rounded to a specified number of precision digits, which is the number of digits after the decimal point of the effective mass in scientific notation.

Parameters:

  • context (Context) – The context at which this collective variable’s effective mass should be evaluated

  • digits (int | None) – The number of precision digits to round to. If None, the value will not be rounded.

Return type:

The effective mass of this collective variable at the given context

Example

>>> import cvpack
>>> import openmm
>>> from openmmtools import testsystems
>>> model = testsystems.AlanineDipeptideImplicit()
>>> peptide = [
...     a.index
...     for a in model.topology.atoms()
...     if a.residue.name != 'HOH'
... ]
>>> radius_of_gyration = cvpack.RadiusOfGyration(peptide)
>>> model.system.addForce(radius_of_gyration)
6
>>> platform =openmm.Platform.getPlatformByName('Reference')
>>> context =openmm.Context(
...     model.system,openmm.VerletIntegrator(0), platform
... )
>>> context.setPositions(model.positions)
>>> print(radius_of_gyration.getEffectiveMass(context, digits=6))
30.94693 Da
